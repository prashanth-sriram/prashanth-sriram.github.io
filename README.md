# Flying With Data: A Data Science Blog


You have to learn to crawl before you can walk.

Learn to walk before you can run.

Learn to run before you can soar.

Learn to soar before you can fly.


This is a personal blog documenting what I learn and practice in order to get better as a data scientist, to reach and stay in the zone where I feel like I am flying. Who knows, maybe it will help others as well.
